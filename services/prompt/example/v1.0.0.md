# Prompt: Resume Keyword Extraction (Structured JSON) - v1.0.0

## Purpose
- 입력된 이력서 텍스트에서 핵심 정보를 구조화(JSON)한다.
- 결과는 DB 저장 및 질문 생성 파이프라인의 입력으로 사용한다.

## Inputs
- resume_text: 이력서에서 추출한 전체 텍스트(UTF-8)

## Output Format (STRICT JSON ONLY)
아래 스키마를 반드시 만족하는 JSON만 출력한다. 설명 문장/마크다운을 출력하지 않는다.

{
  "position": {
    "target_role": "string",
    "level": "string",
    "years": "number"
  },
  "career": [
    {
      "company": "string",
      "role": "string",
      "period": {
        "start": "YYYY-MM",
        "end": "YYYY-MM or PRESENT"
      },
      "highlights": ["string"]
    }
  ],
  "skills": {
    "languages": ["string"],
    "frameworks": ["string"],
    "databases": ["string"],
    "infra": ["string"],
    "tools": ["string"]
  },
  "education": [
    {
      "school": "string",
      "major": "string",
      "degree": "string",
      "period": {
        "start": "YYYY-MM",
        "end": "YYYY-MM or PRESENT"
      }
    }
  ],
  "projects": [
    {
      "name": "string",
      "summary": "string",
      "tech_stack": ["string"],
      "my_role": ["string"],
      "impact": ["string"]
    }
  ],
  "certificates": [
    {
      "name": "string",
      "issuer": "string",
      "date": "YYYY-MM"
    }
  ]
}

## Rules
- resume_text에 없는 정보는 추측하지 말고 빈 문자열("") 또는 빈 배열([])로 둔다.
- years는 숫자 타입으로만 출력한다(예: 0, 1.5, 3).
- period는 가능한 경우만 채운다. 불명확하면 ""로 둔다.
- skills는 중복 제거 후 정규화(대소문자/공백 정리)한다.
- 결과 JSON 이외의 텍스트를 절대 출력하지 않는다.

## Example Input
resume_text:
"""
- Java/Spring Boot 기반 백엔드 개발
- MySQL, Redis, Docker 사용
- 프로젝트: 모의면접 서비스, STT 분석 파이프라인 구축
"""

## Example Output
{
  "position": { "target_role": "Backend Developer", "level": "Junior", "years": 0 },
  "career": [],
  "skills": {
    "languages": ["Java"],
    "frameworks": ["Spring Boot"],
    "databases": ["MySQL", "Redis"],
    "infra": ["Docker"],
    "tools": []
  },
  "education": [],
  "projects": [
    {
      "name": "모의면접 서비스",
      "summary": "이력서 기반 질문 생성 및 STT 기반 답변 분석 파이프라인",
      "tech_stack": ["Java", "Spring Boot", "MySQL", "Redis", "Docker"],
      "my_role": [],
      "impact": []
    }
  ],
  "certificates": []
}
